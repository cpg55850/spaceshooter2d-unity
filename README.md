# 2D Space Shooter

This is a top-down space shooter type game I made in Unity during my last year in undergrad. It is a simple game, but my focus was on learning techinques for code organization and architecture in Unity (for example, ScriptableObjects and Unity Delegates / Events). Also, I wanted to see what it would be like to build a game in 2D in Unity, since I mostly had worked in 3D in Unity up to that point.

You can play the game here: https://charliechip95.itch.io/space-shooter-2d
